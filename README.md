  ## How to run:
 
 instalar as depedencias 
``` bash 
npm i 
```
caso queira rodar no heruko
``` bash 
npm run start 
```
Para rodar na sua maquina, crie o banco de dados:
dump.sql 
e rode o comando
``` bash 
npm run start:dev 
```
Para rodar os teste
``` bash 
npm run test
```
